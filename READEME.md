🚀 PrepWise – AI-Powered Interview Preparation Platform

PrepWise is a web application that helps candidates prepare smarter for interviews by analyzing job descriptions, resumes, and personal input to generate a tailored interview strategy.

---

✨ Features

- 📄 Upload Resume (PDF) or provide self-description
- 🎯 AI-generated match score with job role
- ❓ Technical and behavioral interview questions
- 📊 Personalized preparation strategy
- ⚡ Fast and simple user experience

---

🛠 Tech Stack

Frontend

- React.js
- SCSS
- Vite

Backend

- Node.js
- Express.js

AI Integration

- Google Gemini API

---

⚙️ How It Works

1. User provides:
   
   - Job Description
   - Resume OR Self Description

2. AI processes the input

3. System generates:
   
   - Match Score
   - Interview Questions
   - Preparation Plan

4. Provide ATS Friendly AI Generated Resume for the Job

---

📦 Installation

# Clone the repository
git clone https://github.com/YOUR_USERNAME/prepwise.git

# Navigate to project
cd prepwise

# Install dependencies (frontend)
cd Frontend
npm install
npm run dev

# Open new terminal → backend
cd Backend
npm install
npm start

---

🔐 Environment Variables

Create a ".env" file inside Backend:

GOOGLE_GENAI_API_KEY=your_api_key_here
